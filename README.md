# SchoolManagementSystemFrontend
School Management System (Frontend)
Overview
The School Management System (Frontend) is a responsive website built using
HTML and CSS. It provides a structured interface for students, teachers, and
administrators to interact with the system. This version includes only the frontend
pages; backend functionality (Java Servlets + JSP + MySQL) will be added later.
Project Structure
SchoolManagementSystemFrontend/
│── index.html # Home Page
│── about.html # About Page
│── contact.html # Contact Us Page
│── login.html # Login Page
│── register.html # Registration Page
│── dashboard.html # Dashboard Page
│── css/style.css # Central stylesheet
Features
Home Page (index.html) o Welcome message
o Navigation links to all pages  About Page (about.html)
o Description of the system and its purpose
Contact Us Page (contact.html)
o Contact form with fields: Name, Email, Message
Login Page (login.html)

Register Page (register.html)

Dashboard Page (dashboard.html)
o Placeholder tables for Students and Teachers
o Sample dummy data for demonstration
CSS Styling (style.css)
o Responsive design
o Navigation bar, forms, tables, and footer styling
Future Enhancements  Backend Integration: Java Servlets + JSP + MySQL for authentication, CRUD operations, and

Technologies Used  HTML5 → Structure of pages  CSS3 → Styling and responsiveness  Java (Servlets/JSP) → Planned backend
